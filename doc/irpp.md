# Impot sur le revenu des personnes physiques

Formulaire N°2042

## Revenu net global

total des revenus impossables, ensemble des revenus brut

## Revenu net global impossable

1- Somme des revenus nets catégoriels: salaires, pensions, rentes viagères, revenus fonciers, BIC
2- Déduction forfaitaire de 10% sur les salaires
3- Retrancher deficits foncier et agricoles, certaines charges, ...


## Deficit foncier

Revenu foncier = Loyer - charges > 0
Deficit foncier = loyer - charges < 0

Exemple 1:
Salaire 30K/an
Revenuf foncier: 12K/an
Interet emprunt: 2300/an
Taxe Fonciere: 1000
Assurance: 600
Travaux: 40000

Annee 1
Charges = 2300 + 1000 + 600
Revenu foncier impossable = 12K - charges - travaux = -31900
revenu impossable = 30K - min(10700, 31900) = 19300
deficit reportable = 31900 - 10700 = 21200

Annee 2
Revenu foncier impossable = 12K - charges - deficit reportable = -13100
revenu impossable = 30K
deficit reportable = 13100

Annee 3
Revenu foncier impossable = 12K - charges - deficit reportable = -5000
revenu impossable = 30K
deficit reportable= -5000

Annee 4
Revenu foncier impossable = 12K - charges - deficit reportable = 3100
revenu impossable = 30K + 3100
deficit reportable= 0

Annee 5
Revenu foncier impossable = 12K - charges - deficit reportable = 8100
revenu impossable = 30K + 8100
deficit reportable= 0
